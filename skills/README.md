# Skills

This folder contains reusable skills for reviewing and evaluating Stellar Community Fund (SCF) applications.

## Available Skills

| Skill | Description |
|-------|-------------|
| [scf-reviewer](scf-reviewer.md) | Review SCF Build Award applications across Integration and Open tracks. Covers project fit, technical architecture, team readiness, traction, budget analysis, and funding recommendations. |

## Usage

Each skill is defined as a Markdown file with YAML frontmatter (`name` and `description`) followed by the full instruction set. Point your AI tool or workflow at a skill file to load its review framework.

## Adding a New Skill

1. Create a new `.md` file in this folder.
2. Include YAML frontmatter with `name` and `description`.
3. Write the skill body using the same structure: overview, process, evaluation criteria, output formats, red flags/strengths, and references.
4. Update the table above.
